# AutoCAD Floor Plan Design for Residential Buildings (UK)

## Overview
This project focuses on the creation of architectural floor plans for residential buildings in the United Kingdom using AutoCAD.

The work emphasizes architectural layout planning, spatial organization, and technical drawing standards. All designs were developed independently. Structural (static) calculations are not included.

## Features
- Detailed residential floor plan design
- Architectural layout and space optimization
- Accurate dimensioning and annotations
- Clean and professional technical drawings
- UK-oriented planning context

## Tools & Technologies
- AutoCAD

## Project Scope
- Residential building floor plans
- Room layout and functional zoning
- Technical drafting according to common standards
- Preparation of construction-ready drawings (excluding structural engineering)

## Limitations
- No structural (static) calculations
- Not a substitute for certified architectural or engineering approval

## Usage
This repository is intended for:
- Portfolio presentation
- Showcasing AutoCAD drafting skills
- Demonstrating architectural planning capabilities
